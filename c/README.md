# 🛵 DAKOP

A community-driven navigation tool for motorcyclists and vehicle drivers to avoid checkpoints (e.g., “dakop” operations) when they have expired documents or other license issues. Users can mark checkpoint locations on a map and suggest alternative routes to bypass them.

## ✨ Features

- 🗺️ **Interactive Map** – View real-time checkpoint locations marked by other users.
- ⚠️ **Add Checkpoints** – Signed-in users can mark new “dakop” or checkpoint spots.
- 🔁 **Alternative Routes** – Suggest and view alternate paths to avoid marked checkpoints.
- 🔐 **User Authentication** – Sign up / Sign in required to contribute new checkpoints or routes.
- 🧭 **Community Powered** – Relies on user reports to keep data accurate and up‑to‑date.

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript  
- **Routing**: React Router DOM v6  
- **Styling**: Tailwind CSS + Autoprefixer + PostCSS  
- **Build Tool**: Vite  
- **Icons**: Lucide React  
- **Linting**: ESLint (with TypeScript support)
